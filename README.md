## ghost cms를 이용한 개인 블로그 페이지

[aws lightsail](https://lightsail.aws.amazon.com/)에 ghost cms 서버를 구축하고
[vercel](https://vercel.com/)로 [next.js](https://nextjs.org/) 프론트 페이지를 띄웠다.

스타일링은 [chakra-ui](https://chakra-ui.com/)를 사용했다.
